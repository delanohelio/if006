# Pratica 2026-03-26 - Roteiro do Estudante

## Objetivo

Nesta atividade voce vai inspecionar trechos de codigo Java sem executar.

Para cada pagina, voce deve responder duas perguntas simples.

## Escolha do grupo

- Final do CPF par: use o Grupo A
- Final do CPF impar: use o Grupo B

Total da atividade: 10 questoes (2 por pagina, 5 paginas).

## Como responder

1. Abra a página: [https://delanohelio.github.io/if006/inspecao-codigo/]
2. Leia com calma e nao execute o codigo.
3. Responda cada pergunta.
4. No final copie o resumo e cole no formulario de respostas (que está disponível no final da página da atividade).


## Importante

- O objetivo é leitura e compreensao de codigo.
- Compare com colegas apenas depois da coleta.
